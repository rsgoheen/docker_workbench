An Introduction to Docker

What is Docker?

Docker is not Virtualization

Isolated of Runtime Environment

Encapsulation and separation of:

  • Processes
  • Environmental Variables
  • Filesystem

The Container Timeline

  • Early *nix - user and file isolation (chroot)
  • 2000 - BSD Jails
  • 2004 - Solaris Zones
  • 2006 - Linux cgroup (control groups)
  • 2013 - Docker
  • June 2014 - Docker Version 1.0 Release

Docker == containers

Containers != Docker

But What is Docker?

A set of tools and a file format for describing containers and a runtime for setting up and running isolation environments on an existing operating system.

  • Open source
  • Written in Go
  • Docker, Inc. (previously dotCloud)

Docker's Popularity

  • GitHub project: 1,500 contributors, 10,000 forks, 35,000 stars
  • Large ecosystem
  • Backing by Microsoft

... but why?

  • Abhorring the vaccuum
  • Microservices paradigm
  • Cattle, not pets
  • Infrasctructure as code
  • Docker provided the best, simplest container implementation available
