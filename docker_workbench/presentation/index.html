An Introduction to Docker

What is Docker?

Docker is not Virtualization

[graphic of virtualization]

Docker is an Isolated Runtime Environment

Isolation of:

  • Processes
  • Filesystem
  • Environmental Variables
The unit of isolation is known as a container. However, containers are often referred to in a more generic context and not necessarily Docker. Containers are an encapsulation of an application with its dependencies. Containers share resources. Containers do not require the full start up cycle of a VM or OS

The Container Family Tree

  • Early *nix - user and file isolation (chroot)
  • 2000 - BSD Jails
  • 2004 - Solaris Zones
  • 2006 - Linux cgroup (control groups)
  • 2013 - Docker

But What is Docker?

A set of tools and file format for describing containers and a runtime for setting up and running isolation environments on an existing operating system.
